About FrostedEngineering
FrostedEngineering is a Minecraft plugin framework designed to create Minecraft mod like experience in Spigot/Bukkit/Paper.
The plugin offeres a modular API system, which allows you to create/install new "mods".

What does FrostedEngineering offer?

Modularity

Want to disable or just enable more features?
You can simply install modules to add even more features.

Easy-to-use

Both the installing and setting up the plugin is very easy.
We kept things user-friendly and understandable.

Frequent Updates

The plugin receives updates about every single week.
These could be minor or even major fixes/updates.



Powerful API

Are you a developer and you want to access our features?
You're in the right place, we have an API built in for you.

Unique Experience

This plugin offeres a very advanced mod experience but in Vanilla!
Try it out today!.

Open Source
